# TL Tide BT Dashboard

Team Lead dashboard for Tide BT operations.

## Setup

```bash
npm install
npm start
```

Runs on port 3005.

## Features

- TL Profile
- View FSEs under TL (from Tide BT Access)
- Tide BT specific features (coming soon)

## Backend

- Port 4000: Authentication & Profile (existing Tide backend)
- Port 4002: Tide BT TL specific features (to be built)
